# iRevolution-A-Data-driven-Exploration-of-Apple-s-iPhone-Impact-in-India-using-Tableau
Smartbridge Project :::: iRevolution : A data-driven exploration of Apple's iPhone impact in India using Tableau.
 The project visualizes smartphone trends, market share, and consumer preferences.

 It highlights the evolution of iPhone models and key technological innovations.

 Interactive dashboards present model-wise share and brand competition.

 Country-wise analysis compares smartphone preferences across regions.

💰 Revenue trends show Apple’s growth and premium market positioning.

 Battery type distribution explains usage of Lithium-Ion and Lithium-Polymer technologies.

 Price comparison across brands reflects market segmentation and affordability factors.

 🇮🇳 The project emphasizes India’s value-driven smartphone market behavior.

 Tableau dashboards enable clear and interactive data storytelling.

 The web interface is built using Flask and Bootstrap template integration.
